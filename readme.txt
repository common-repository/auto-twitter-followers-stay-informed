Automatically tweets each new blog post directly to twitter. Get Top backlinks, inform your followers in real time, all on autopilot.

== Description ==
Plugin adds a check box to the Write Post editor. When this box is checked, plugin automatically tweets each new blog post with a link of course.
Maximum advantage of simplicity. In the WP admin interface, set necessary keys.
Automatic tweet an older articles with one click.
If a post is longer than 140 characters, it will be cut down. For most of this area did not take a link to your site, supports plug and Shorten Service bit.ly
Auto twitter followers stay informed plugin put your social media to autopilot.
== Installation ==
    * Download the plugin package and expand it.
    * Upload WP_Auto_Tweet_Plugin to your plugins directory, likely wp-content/plugins/.
    * Activate the plugin through the Plugins menu in WordPress.
    * Visit the Auto Tweet plugin options page to input your Twitter username and password
